## Installatation and usage
Clone repository:
```
git clone https://github.com/ElliotMS-Chalmers/shogi.git
cd shogi 
```
### Windows
Build and run using included bat script.
```
./gradlew.bat run
```
### Linux
Build and run using included bash script.
```
./gradlew run
```
